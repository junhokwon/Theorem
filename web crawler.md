# 웹 크롤러

: 조직적,자동화된 방법으로 월드와이드웹(WWW)을 탐색하는 컴퓨터 프로그램이다. 

* pyenv 가상환경을 새로 설정해줘야한다.

(1) 가상환경생성 : pyenv virtualenv '버전명' '폴더이름'

(2) 가상환경을 해당 폴더에서 사용하도록 지정

`pyenv local '가상환경폴더이름`


* pycharm에서 가상환경을 적용해줘야 한다.
